# Расчет вентиляции в горячем цехе и обеденном зале кафе

## Описание

Этот Excel файл содержит полный расчет системы вентиляции для кафе на основе **Примера 5.1** из **Сборника расчетов АВОК-2020**.

## Файлы в репозитории

- **Расчет_вентиляции_кафе.xlsx** - готовый Excel файл с расчетами и формулами
- **create_ventilation_calc.py** - Python скрипт для генерации Excel файла
- **123123123 и воздухообмена в помещениях жилых и общественных зданий (Сборник расчетов АВОК)-2020.pdf** - исходный документ с примером
- **Dokument-Microsoft-Word-2.docx** - документ Word

## Структура Excel файла

### Лист 1: "Расчет вентиляции"

#### 1. Исходные данные
- Количество посадочных мест: **20**
- Численность персонала в горячем цехе: **3 человека**
- Объем горячего цеха: **90 м³** (высота 3,1 м)
- Объем обеденного зала: **331,2 м³** (высота 3 м)
- Коэффициент одновременности работы оборудования: **0,7**
- Коэффициент эффективности местного отсоса: **0,7**
- Мощность освещения горячего цеха: **0,71 кВт**
- Температуры воздуха: отсосы **42°C**, верхняя зона **30°C**, обеденный зал **27°C**

#### 2. Таблица 5.1 - Оборудование горячего цеха
Перечень из 8 единиц оборудования с характеристиками:
- Пароконвекционные печи (2 типа)
- Холодильный прилавок
- Гриль
- Станция напитков
- Охладитель молока
- Льдогенератор
- Слайсер

Для каждой позиции автоматически рассчитывается **Qу·Kз**

#### 3. Расчеты по формулам

**Формула (1)** - Тепловыделения от кухонного оборудования:
```
Q₁ = Kо·[(∑Qу·Kз·(1-Kэфобщ)) + (∑Qу_р·Kз)]
Результат: Q₁ = 12,53 кВт
```

**Формула (2)** - Тепловыделения от людей:
```
Q₂ = n₂ · q₂
Результат: Q₂ = 0,6 кВт
```

**Формула (3)** - Общие тепловыделения в горячем цехе:
```
Qг = Q₁ + Q₂ + Q₃
Результат: Qг = 13,84 кВт
```

**Формула (4)** - Воздух, удаляемый из верхней зоны:
```
Lв = n · Vг
Результат: Lв = 180 м³/ч
```

**Плотности воздуха:**
- ρᵢ (для местных отсосов при 42°C) = **1,13 кг/м³**
- ρв (для верхней зоны при 30°C) = **1,17 кг/м³**
- ρрт (для обеденного зала при 27°C) = **1,18 кг/м³**

**Формула (5)** - Массовый расход удаляемого воздуха:
```
Gуг = (∑Lоᵢ · ρᵢ) + Lв · ρв
Результат: Gуг = 7440 кг/ч
```

**Распределение воздушных потоков:**
- Gс (через раздаточный проем) = **1490 кг/ч** (20% от Gуг)
- Gпог (приточный воздух в горячий цех) = **5950 кг/ч** (80% от Gуг)

**Формула (12)** - Объемный расход через раздаточный проем:
```
Lс = Gс / ρрт
Результат: Lс = 1260 м³/ч
```

#### 4. Итоговые результаты

Таблица с 11 ключевыми параметрами и контрольными значениями из PDF для проверки.

### Лист 2: "Справочные таблицы"

- **Таблица 5.4** - Количество теплоты, выделяемое взрослыми мужчинами (в зависимости от температуры и характера работы)
- **Таблица 5.5** - Поправка на положение источника теплоты (свободно стоящее, у стены, в углу)

## Особенности реализации

### ✓ Все формулы реализованы в Excel
- При изменении исходных данных все результаты автоматически пересчитываются
- Используются функции SUMIF для фильтрации оборудования с отсосами и без

### ✓ Соответствие примеру из PDF
- Все расчеты выполнены в точном соответствии с методикой из Сборника расчетов АВОК-2020
- Результаты совпадают с примером 5.1
- Использованы те же округления, что и в оригинале (например, ρᵢ = 1,13 кг/м³)

### ✓ Удобное оформление
- Цветовое выделение заголовков и результатов
- Границы таблиц для лучшей читаемости
- Итоговые результаты выделены желтым цветом
- Контрольные значения из PDF для проверки

## Как использовать

1. Откройте файл **Расчет_вентиляции_кафе.xlsx** в Microsoft Excel или LibreOffice Calc
2. Все формулы автоматически пересчитаются
3. При необходимости измените исходные данные в разделе "ИСХОДНЫЕ ДАННЫЕ"
4. Результаты обновятся автоматически
